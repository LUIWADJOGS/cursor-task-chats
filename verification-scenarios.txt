# Cursor Task Chats — сценарии проверки

## Deeplink / prompt
1. Выполнить "Create Task", ввести title и при необходимости описание.
2. Должен открыться новый чат Cursor с префиллом: ветка, задача, статус, подзадачи (если есть), прогресс по файлам/коммитам от baseline, инструкции.
3. URL должен быть вида cursor://anysphere.cursor-deeplink/prompt?text=...

## Хранение
1. Создать задачу на ветке feature/x (после создания появится привязанный чат).
2. Перезапустить Cursor.
3. Открыть панель "Task Chats" (иконка чеклиста в Activity Bar) — ветка и задачи для feature/x должны отображаться.
4. Кликнуть по чату под задачей — должен открыться существующий чат Cursor.
5. "Remove Chat From Task" — чат исчезает из дерева; "Archive Task" — задача уходит в архив и не показывается.

## Фильтрация по ветке
1. На ветке feature/x создать задачу с чатом.
2. Переключиться на main (или другую ветку).
3. В дереве отображаются только задачи для текущей ветки (для main — свои или пусто).
4. Вернуться на feature/x — снова видны задачи этой ветки.
5. При переключении ветки (если есть корневые задачи) — уведомление: "Branch: <name>. Root tasks: N."

## Команды
- "Show Task Chats For Current Branch" — Quick Pick по всем чатам задач на ветке; выбор открывает чат.
- Контекст по задаче: новый чат задачи, подзадача, копирование prompt задачи, статус, диффы, архив.
- Контекст по чату: Open, Copy Prompt, диффы, переименование, убрать из задачи.
